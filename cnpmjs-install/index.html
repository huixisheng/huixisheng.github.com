发表于 2018-03-27   |  

cnpmjs 可用于企业的前端私有包管理。淘宝NPM镜像就是用这个搭建的。

npmjs私有包是要收费的，对于公司而言更倾向于自己部署，一是省钱，二是一些含有敏感内容的包放到其他平时总是不放心。

本文将简单记录下实际安装过程中碰到的问题，基本的安装可以移步官方文档。
